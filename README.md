# redbreast

Telegram bot for Stonks.

## Usage

    $ lein uberjar
    $ java -cp target/uberjar/redbreast.jar clojure.main -m redbreast.core

## Libraries

- [morse][0]
- [clj-http][1]
- [cheshire][2]
- [clj-xchart][3]

[1]: https://github.com/dakrone/clj-http
